# Wardrobe Redesign Tool

## Purpose
Redesign wardrobes and built-in storage visually and strategically, including:
- shutter style
- panel rhythm
- laminate finish
- handle strategy
- mirrored section options
- open niche / accent section ideas

## Tool modes
1. finish-only wardrobe refresh
2. shutter style redesign
3. premium wardrobe concept variant generation

## AURA responsibilities
- identify wardrobe type and style context
- recommend shutter language (flat, fluted, framed, mirrored, etc.)
- recommend laminate/color direction
- recommend handle/profile strategy
- warn about over-busy panelization
- critique realism and fit with room palette

## Validation ideas
- preserve wardrobe footprint
- avoid impossible panel seam logic
- preserve hardware zones where requested
- preserve mirrors only when requested

## Premium UI notes
- shutter style gallery
- handle strategy selection
- panel rhythm preview
- wardrobe concept variants as cards
- “minimal / elegant / luxe / warm modern” presets

## Future enhancement
- internal storage planning mode
- wardrobe module segmentation if layout data exists
