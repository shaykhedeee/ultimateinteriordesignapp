# Curtain Changer Tool

## Purpose
Change curtains/drapes/blinds/shades while preserving:
- window geometry
- rod/track placement if locked
- daylight logic
- fold behavior
- material believability

## Supported categories
- sheer curtains
- blackout drapes
- ripple fold curtains
- roman blinds
- roller blinds
- venetian / slatted systems (advanced)

## AURA responsibilities
- infer suitable curtain type from room style and function
- suggest color/material opacity
- warn about over-heavy fabric or wrong formality level
- critique daylight and fabric realism

## Validation ideas
- spill beyond window area
- unrealistic fold geometry
- lighting mismatch near window
- transparency mismatch for sheers

## Premium UI notes
- fabric swatches + translucency preview
- style categories: soft, tailored, luxe, minimal
- daylight impact note

## Future enhancement
- open/closed state variants
- day/night mode comparison
