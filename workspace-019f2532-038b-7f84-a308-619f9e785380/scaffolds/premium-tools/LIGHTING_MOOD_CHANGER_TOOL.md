# Lighting Mood Changer Tool

## Purpose
Adjust lighting mood, warmth, intensity, atmosphere, and time-of-day feel without changing geometry.

## Supported moods
- natural daylight
- warm evening
- luxury ambient
- soft hospitality
- cool gallery
- cinematic dramatic
- task-lit clean modern

## AURA responsibilities
- recommend lighting mood by room type and style
- describe likely impacts on materials and perceived space
- warn if requested mood conflicts with scene logic
- critique realism after generation

## Validation ideas
- overexposed highlights
- implausible shadow directions
- inconsistent color temperature zones
- washed-out or muddy material appearance

## Premium UI notes
- mood chips with visual thumbnails
- intensity/warmth sliders
- day/evening compare states
- “how this changes perception” note from AURA

## Future enhancement
- separate ambient/accent/task controls
- luminaire-aware lighting planning if catalog supports it
