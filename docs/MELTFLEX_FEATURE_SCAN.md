# MeltFlex Feature Scan

## Core value proposition
MeltFlex combines 3 workflows: photo redesign, floor plan to 3D, and real product-furnished exports with purchase links.

## Features to match or exceed
1. **Real furniture purchasing**: Uses real product IDs/carts from IKEA etc. We will not copy their catalog, but we should support factory modular items with user-set pricing.
2. **Price-per-item visible in design flow**: Shows estimated cost while placing furniture, not only at checkout.
3. **Style-aware cost bands**: Same room at economy/premium/luxury tiers with clear price diffs.
4. **Floor plan to interactive 3D**: Successful state. Our 3D scene, walkthrough, and 360 viewer already exceed this.
5. **Photo redesign from floor plan output**: Render from CAD → AI render; exists, needs tighter tool wiring and real provider execution.
6. **Fast defaults**: Beginner-friendly upload, auto-style, auto-furnish. We should add quick-start template projects.

## What MeltFlex does NOT have
- Indian interiors expertise, Vastu, pooja units, Merino laminates
- Factory CNC/DXF/cutlist
- Agentic AI memory and self-learning
- Vendor approval + payment milestone + BOM workflows
- Offline-first architecture

## Strategy to beat them
- Out-execute on end-to-end Indian factory workflow.
- Add a **settings-driven pricing engine** with price per sq ft, hardware adders, labor bands, markup profiles, and client/ex-showroom toggle.
- Ensure every generation tool returns price impact alongside the visual result.
