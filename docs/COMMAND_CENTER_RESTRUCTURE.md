# Command Center Restructure

## Goal

Turn the command center into the operating dashboard for the whole studio.

## Structure

- Top band: project selector, active stage, render/elevation readiness, and one-click next actions.
- Middle band: workflow workspaces grouped by the canonical flow.
- Right rail: task queue, project health, and latest outputs.
- Bottom band: specialist tools, approvals, and job status.

## Required Grouping

- Intake and brief.
- CAD and plan intelligence.
- Scene and modular design.
- Renders and material swap.
- Elevations and cutlists.
- Finance and delivery.

## Behavior

- Show only the tools relevant to the current stage by default.
- Keep the next best action visible.
- Make status transitions obvious when a user finishes a stage.
