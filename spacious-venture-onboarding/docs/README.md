# Spacious Venture Product Documentation

This folder is the planning source of truth for the next version of the Spacious Venture app.

## Product Decision

The visual end goal is the premium dark command-center dashboard shown in the user reference: black sidebar, gold accents, dense project table, right-side readiness panels, and operational dashboard feel.

The functional scope is narrowed to what Spacious Venture actually needs:

1. Fast client onboarding and project intake.
2. Accurate, floor-plan-aware AI render variants for the first client reveal.
3. Clean PDF design brief/proposal output.
4. Cutlist project workflow for modular furniture production.
5. Project status, files, exports, correction memory, and reusable image/standards libraries.

AI render generation is now a core module. The app should work as a digital experience centre: capture requirements, turn layout markers and furniture needs into controlled render options, let the designer select/correct the best option, and carry approved scope into the PDF brief and cutlist workflow.
