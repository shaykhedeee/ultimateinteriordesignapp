# Indian Interior Reference Library & Image Collection

## Purpose

This library serves as both:
1. **Training data** for the AI render engine — teaches it Indian aesthetics, furniture, materials
2. **Inspiration references** for designers and clients during onboarding
3. **Style examples** that can be used as "reference images" in the prompt compiler

## Reference Image Categories & What They Teach the AI

### Living Rooms

| Sub-category | What the AI learns | Example styles |
|-------------|-------------------|----------------|
| Modern Living | Sofa placement, TV unit proportions, false ceiling patterns | Clean lines, neutral tones, accent wall |
| Traditional Living | Wooden furniture, ethnic decor, warm lighting | Carved furniture, brass accents, silk cushions |
| Minimalist Living | Space optimization, clutter-free, natural light | White walls, indoor plants, light wood |
| Luxury Living | Statement pieces, premium finishes, layered lighting | Chandeliers, marble, velvet upholstery |
| Open Plan Living | Kitchen-dining-living flow, zone separation | Rugs as dividers, peninsula counters, pendant lights |
| Compact Living | Small space solutions, multi-functional furniture | Wall-mounted desks, foldable dining, loft storage |

**Key elements to capture**: Sofa type (L-shape vs 3-seater), coffee table style, TV unit design, flooring material, window treatments (curtains/blinds), accent wall treatment, lighting fixtures

### Kitchens

| Sub-category | What the AI learns | Example layouts |
|-------------|-------------------|----------------|
| Modular Kitchen | Cabinet layout, counter space, appliance placement | L-shape, U-shape, Parallel, Straight |
| Indian Kitchen | Spice rack, kadai storage, masala drawer, wet area | Traditional cooking zone, ventilation |
| Open Kitchen | Living-connected, island/peninsula, breakfast counter | Bar stools, open shelving, range hood |
| Premium Kitchen | Quartz counter, soft-close drawers, acrylic finish | Handleless cabinets, LED under-cabinet |

**Key elements**: Counter material (granite/quartz/laminates), backsplash, cabinet finish (acrylic/PU/laminate), hardware (brass/chrome/matte black), chimney type, sink position (near window)

### Bedrooms

| Sub-category | What the AI learns |
|-------------|-------------------|
| Master Bedroom | Wardrobe placement, bed positioning (vastu), bed design (upholstered/wooden), side tables, lighting |
| Guest Bedroom | Space-efficient layout, multi-purpose furniture, study corner |
| Kids Bedroom | Bunk beds, storage for toys, study table, fun colors |
| Small Bedroom | Loft storage, sliding wardrobe, compact furniture, light colors |

### Wardrobes

| Sub-category | What the AI learns |
|-------------|-------------------|
| Sliding Wardrobe | Door design, internal layout (hanging/shelves/drawers), laminate finish |
| Walk-in Wardrobe | Layout planning, island unit, shoe storage, dressing mirror |
| Hinged Wardrobe | Classic design, handle styles, crown moulding |
| Modern Wardrobe | Handleless, push-to-open, LED strip lighting, glass shutters |

**Key elements**: Internal layout (hanging rods ×2 levels, shelves, drawers), door finish, handles, mirror integration, lighting

### Pooja / Mandir Units

| Sub-category | What the AI learns |
|-------------|-------------------|
| Wall-mounted | Compact design, diya shelf, drawer for pooja items, brass accents |
| Floor-standing | Larger unit, multiple tiers, OM symbol, marble top |
| Corner Unit | Space-saving, angled design, soft lighting |
| With Partition | Room divider + mandir, carved jali work |

### TV Units

| Sub-category | What the AI learns |
|-------------|-------------------|
| Wall-mounted | Floating design, cable management, ambient lighting, display shelves |
| Floor-standing | Credenza style, storage + display, contrasting finishes |
| Full-wall | Floor-to-ceiling, library integration, accent panel |
| Corner TV Unit | Space-optimized, angled viewing, compact |

## Using the Library in the App

### During Onboarding (Client References)
```
Client: "I want something like this..."
Designer: Opens reference library → shows 6 images
Client: "I like #3 and #5"
→ Images #3 and #5 are stored as "style references" in the project
→ AI uses these as additional context in the render prompt
→ "Generate a living room inspired by reference image style"
```

### During AI Render Generation
```
The reference library is used in TWO ways:

1. Style matching:
   - AI analyzes the reference image's: color palette, furniture style,
     lighting mood, material finishes
   - These are extracted and added to the compiled prompt:
     "Style reference: warm beige walls, walnut wood tones, 
      teal and mustard accent colors, modern minimalist furniture"

2. Prompt injection:
   - The most similar reference image from the library is found
   - Its metadata is used to enrich the prompt with specific details
   - Ensures the AI generates INDIAN interiors, not Western defaults
```

### During Color Selection
```
Designer selects "TV Unit Material"
→ Library shows laminate swatches from Merino, Greenlam, etc.
→ Also shows PREVIOUS PROJECTS where this laminate was used
→ "Here's how this laminate looked in 3 other projects"
```

## Collection Method

To populate the library:
1. Use free stock photography sites (Pexels, Pixabay, Unsplash — filter "Indian home interior")
2. Partner with Indian interior design brands (Livspace, DesignCafe, Homelane — with permission)
3. Generate synthetic data from the AI itself (best results → add to library)
4. Client project archives (actual completed projects, with permission)

**All images must be:**
- Free for commercial use OR properly attributed
- High resolution (1920×1080 minimum)
- Well-labeled with accurate metadata
- Representative of real Indian homes (not just show homes)
