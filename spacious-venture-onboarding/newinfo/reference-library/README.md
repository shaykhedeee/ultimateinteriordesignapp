## How To Add Images

1. Place high-resolution JPG images in the appropriate category folder
2. Add metadata entry in `metadata.json` following the existing schema
3. Key metadata fields: id, filename, style, budgetTier, keyFeatures, source

## Image Sources (Recommended)

- **Unsplash** (free, no attribution required): Search "Indian home interior"
- **Pexels** (free): Search "Indian interior design" or "modern living room"
- **Pixabay** (free): Search "home interior India"
- **AI Generated**: Use the app's render studio or external AI tools

## Requirements

- Minimum resolution: 1920×1080
- Format: JPG
- Must represent REAL Indian interior styles (not just Western homes)
- Each image should teach the AI something about Indian design: materials, colors, layouts, furniture
