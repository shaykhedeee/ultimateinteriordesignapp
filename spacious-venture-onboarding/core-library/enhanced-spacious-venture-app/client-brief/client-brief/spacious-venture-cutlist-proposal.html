02 — The Algorithm

Multi-Pass Nesting Engine

Upgraded visual layout technology that outperforms standard single-pass nesting tools to compress waste under 10%.

🤖 Multi-Pass Best-Area-Fit (BAF) Routine

Instead of simple linear packing, the engine runs **four concurrent sorting passes** across all panel inputs (Sorting by: Total Surface Area, Width, Height, and Maximum Dimension).

The Best-Area-Fit solver places each cabinet part inside the available board rectangle that minimizes leftover corner spaces, testing rotated axes dynamically (under grain permissions). The engine compares all four packing passes and automatically loads the layout that uses the fewest plywood sheets.

🪵 Woodgrain Constraints Integration

For modular fronts, veneers, and wood-grain laminates, lock grain alignment vertically along the sheet length. For interior carcasses, backing plies, and solid color laminates, disable grain matching to allow free panel rotations and boost nesting efficiency to >93%.
