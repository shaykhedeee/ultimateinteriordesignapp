# Product Brief And Pricing

Date: 2026-06-02

## Product Name

Spacious Venture Studio OS - Phase 1

## Product Promise

Turn every interior client conversation into a structured PDF design brief and a production-ready cutlist project foundation.

The product is not sold as a generic AI image gallery. It is sold as a workflow system for a studio that needs consistent onboarding, cleaner client approvals, and better handoff from design to production.

## Target Buyer

- Interior design studio owner.
- Small to mid-size modular interior firm.
- Designer-led studio that needs professional client documentation.
- Studio that currently depends on WhatsApp notes, Excel sheets, manual PDFs, and disconnected production files.

## Phase 1 Modules

### 1. Command Center

- Dark Spacious Venture dashboard.
- Project pipeline.
- Brief and cutlist readiness indicators.
- Admin activity view.
- Studio quick actions.

### 2. Add Client Onboarding

- Guided intake from the `Add Client` button.
- Client/project details.
- Room and module scope.
- Budget and timeline.
- Floor plan upload.
- Manual floor plan annotations.
- Material and production preferences.

### 3. PDF Design Brief

- Structured brief generator.
- Floor plan preview.
- Room scope and requirements.
- Materials and finishes.
- Site constraints and checks.
- Sign-off section.
- Client-ready PDF export.

### 4. Cutlist Project Foundation

- Create cutlist project from approved brief.
- Module planning screen.
- Production assumptions.
- Board, laminate, edge banding, and hardware defaults.
- Part list structure prepared for Phase 2.

### 5. Materials Library

- Laminate/material shortlist.
- Board and finish notes.
- Practical maintenance notes.
- Indian modular interior assumptions.

### 6. Backup And Handover

- Local SQLite database.
- Local file storage.
- Export/backup route foundation.
- Documentation for future enhancement.

## One-Time Fee

Recommended quote: INR 1,45,000 one-time implementation fee.

This stays below the requested INR 1.5 lakh cap.

## Payment Milestones

- Advance: INR 58,000 at project start.
- Phase 1 working build: INR 58,000 after dashboard, onboarding, and brief workflow are demo-ready.
- Final handover: INR 29,000 after PDF/export verification and documentation.

## Included In The Fee

- Product planning and documentation.
- React/Vite frontend.
- Node/Express backend.
- SQLite database.
- Local file storage for floor plans and PDFs.
- Dashboard and navigation refactor.
- Add Client onboarding flow.
- PDF brief workflow.
- Cutlist project foundation.
- Material library screen.
- Basic QA and handover notes.

## Not Included In The Fee

- Paid API usage for image generation or AI providers.
- Production cloud hosting bills.
- Advanced user roles and permissions.
- DWG/DXF/CAD parser.
- CNC-grade sheet optimizer.
- ERP/accounting/inventory integration.
- Vendor live catalog integration.
- Native mobile app.

## Suggested Sales Positioning

Use this positioning with the client:

"This is a studio workflow app for onboarding, design brief documentation, and modular cutlist preparation. It helps your team collect the right client details, create a professional PDF brief, and move approved scope toward production without losing information."

## Success Criteria

- Admin can click `Add Client` and complete onboarding.
- Floor plan and room data are captured.
- App generates a clean PDF brief.
- Cutlist project can be created from the approved brief.
- Dashboard shows status/readiness.
- The app can be demoed as a polished Spacious Venture product.
