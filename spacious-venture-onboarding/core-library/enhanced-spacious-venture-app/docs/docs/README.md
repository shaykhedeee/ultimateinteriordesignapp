# Spacious Venture Product Documentation

This folder is the planning source of truth for the next version of the Spacious Venture app.

## Product Decision

The visual end goal is the premium dark command-center dashboard shown in the user reference: black sidebar, gold accents, dense project table, right-side readiness panels, and operational dashboard feel.

The functional scope is narrowed to what the client actually asked for:

1. Client onboarding and project intake.
2. Clean PDF design brief/proposal output.
3. Cutlist project workflow for modular furniture production.
4. Project status, files, exports, and reusable standards.

AI moodboard/image generation can remain a future/optional module, but it should not drive the core product unless the client later asks for it.
